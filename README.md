# Nexa Study

Nexa Study is a local MVP of an AI-powered study assistant web app built with Python, Streamlit, and the OpenAI API.

## Features

- Paste study notes
- Choose subject area, difficulty level, and output style
- Generate summaries, flashcards, quiz questions, revision packs, and exam cheat sheets
- Markdown and PDF export in progress

## Tech Stack

- Python
- Streamlit
- OpenAI API
- python-dotenv

## Status

Local prototype. Not deployed yet.
